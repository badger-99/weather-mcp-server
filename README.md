# Weather MCP Server

A **Model Context Protocol (MCP)** server built in **Node.js** and **TypeScript**, based on the [Anthropic MCP tutorial](https://modelcontextprotocol.io/quickstart/server#node). This server is designed to provide contextual weather data to compatible LLM clients via the MCP standard.

> 🚧 **Work in Progress**: This project is still under development. Installation and usage instructions will be added as the project matures.

## Tech Stack

- **Node.js**
- **TypeScript**
- **Model Context Protocol (MCP)**

## Overview

Currently, the project is in its early stages. I'm using it to explore how to convert local MCP servers into remote MCP servers.
